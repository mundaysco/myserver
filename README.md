# My Server Project

## Description
A Node.js server application for Max-Clover

## Installation
```bash
npm install
```

## Setup
1. Copy `.env.example` to `.env`
2. Fill in your environment variables
3. Run `npm start`

## API Endpoints
- GET / - Home page
- GET /api/data - Get data
- POST /api/data - Create data

## Environment Variables
See `.env.example` for required variables

## Development
```bash
npm run dev
```
